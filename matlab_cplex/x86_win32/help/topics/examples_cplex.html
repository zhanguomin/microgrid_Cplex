Examples of the Cplex Class API

These examples illustrate the usage of the Cplex Class API to solve mathematical programming problems.

  • Example: runclassexamples.m
    This function executes all of the provided examples of the Cplex Class API in succession.
  • Example: blend.m
    This example illustrates using the Cplex Class API to model and solve a blending problem.
  • Example: cutstock.m
    This example illustrates using column generation to solve a cutting stock problem.
  • Example: diet.m
    This example illustrates modeling by columns and modeling by rows.
  • Example: facility.m
    This example illustrates using the Cplex Class API to model and solve a facility location problem.
  • Example: fixcost1.m
    This example illustrates using the Cplex Class API to model and solve a production planning problem with fixed costs.
  • Example: indefqpex1.m
    This example illustrates first-order solutions in a quadratic programming problem.
  • Example: inout1.m
    This example illustrates using the Cplex Class API to model and solve a production model.
  • Example: inout3.m
    This example illustrates using the Cplex Class API to model and solve a production model.
  • Example: lpex1.m
    This example illustrates using the Cplex Class API to populate data in a model.
  • Example: lpex2.m
    This example illustrates using the Cplex Class API to add data from a file.
  • Example: lpex3.m
    This example illustrates using the Cplex Class API to add rows to a model and re-optimize.
  • Example: lpex6.m
    This example illustrates using the Cplex Class API to load and use a basis from a file.
  • Example: lpex7.m
    This example illustrates using the Cplex Class API to read in and optimize a problem as well as print names with the answer.
  • Example: mipex1.m
    This example illustrates using the Cplex Class API to enter and optimize a mixed integer programming problem, demonstrating different methods for creating a problem.
  • Example: mipex2.m
    This example solves another basic mixed integer programming problem (MIP) with data from files.
  • Example: mipex3.m
    This example illustrates using the Cplex Class API to enter and optimize a mixed integer programming problem with special ordered sets (SOS), demonstrating different methods for creating a problem.
  • Example: mipex4.m
    This example demonstrates how to use an aborter to interrupt optimization and how to use informational callbacks for logging, timing, or setting a node limit.
  • Example: miqpex1.m
    This example illustrates using the Cplex Class API to create and solve a MIP with a quadratic objective (MIQP).
  • Example: populate.m
    This example illustrates using the Cplex Class API to read in and generate multiple solutions to a mixed-integer programming problem (MIP).
  • Example: qcpex1.m
    This example illustrates using the Cplex Class API to solve a quadratically constrained problem (QCP).
  • Example: qpex1.m
    This example illustrates how to solve a problem with a quadratic objective (QP).
  • Example: qpex2.m
    This example illustrates how to solve a problem with a quadratic objective (QP).
  • Example: rates.m
    This example illustrates how to solve a planning problem with semi-continuous variables.
  • Example: steel.m
    This example illustrates using the Cplex Class API to solve a multiperiod production model.
