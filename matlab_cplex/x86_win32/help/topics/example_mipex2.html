Example: mipex2.m

This example solves another basic mixed integer programming problem (MIP) with data from files.

To run this example, a function argument is required, i.e., mipex2(filename), where filename is the name of the file, with .mps, .lp, or .sav extension.

See also the example file mipex2.m.
