Example: qpex2.m

This example illustrates how to solve a problem with a quadratic objective (QP).

To run this example, a function argument is required, i.e., qpex2(filename), where filename is the name of the file, with .mps, .lp, or .sav extension.

See also the example file qpex2.m.
