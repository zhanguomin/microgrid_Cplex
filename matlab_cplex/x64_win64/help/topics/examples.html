Examples

Illustrative examples of mathematical programming problems that can be solved are provided.

  • Examples of the CPLEX for MATLAB Toolbox
    These examples illustrate the usage of toolbox functions to solve mathematical programming problems.
  • Examples of the Cplex Class API
    These examples illustrate the usage of the Cplex Class API to solve mathematical programming problems.
  • Example: cplexgui.m
    This example uses GUI methods to build an application and use the method cplex.terminate() to interrupt an ongoing optimization.
  • Example: cplexplot.m
    This example illustrates using the Cplex Class API to plot intermediate solutions while solving a problem.
